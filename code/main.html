Mathematical Modeling in GAM

The module is designed to calculate the energy characteristics of atomic and molecular species using nitrogen as an example. The main classes for calculating characteristics are presented in the file "transport.py". The files "N.txt" and "N2.txt" contain spectroscopic data for atomic and molecular nitrogen, they are required for initialization. Usage examples are given below in this file.

def PlotFigure(t, y1N, y2N, y1N2, y2N2, yLabel):
    plt.figure(figsize = (10, 4))
    plt.subplot(1, 2, 1)
    plt.plot(t, y1N, color = 'blue', linestyle = 'solid', linewidth = 2, label = "Translational")
    plt.plot(t, y2N, color = 'red', linestyle = 'solid', linewidth = 2, label = "Internal")
    plt.grid(True)
    plt.xlabel('T, K')
    plt.ylabel(yLabel)
    plt.title('Atomic Nitrogen')
    plt.legend()
    plt.subplot(1, 2, 2)
    plt.plot(t, y1N2, color = 'blue', linestyle = 'solid', linewidth = 2, label = "Translational")
    plt.plot(t, y2N2, color = 'red', linestyle = 'solid', linewidth = 2, label = "Internal")
    plt.grid(True)
    plt.xlabel('T, K')
    plt.ylabel(yLabel)
    plt.title('Molecular Nitrogen')
    plt.legend()    
    plt.show()

An example of calculating specific heats

t = np.linspace(500, 10000, 50)
shTrN = [N.SpecificHeatTr(x) for x in t]
shIntN = [N.SpecificHeatInt(x) for x in t]
